HMI WiFi board
==============

Features
--------

  * STM32F103Cxxx MCU;
  * TFT LCD 3.2" with resistive touchscreen;
  * WiFi ESP8266 based module;
  * USB device;
  * MicroSD card slot;
  * Buzzer;
  * DHT22 themperature/humidity sensor.

Library
-------

  * [ChibiOS/RT](http://www.chibios.org/)
  * [uGFX](http://ugfx.org/)
